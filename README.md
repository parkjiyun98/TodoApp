# TodoApp
Create
1. Todo 목록 등록

Read
1. 등록된 전체 Todo 목록 조회
2. 등록되어있는 Todo의 특정 id를 입력하여 조회

Update
1. 이미 한 일 완료 표시
2. Todo의 내용을 수정

Delete
1. 등록된 전체 Todo 삭제
2. 등록되어있는 Todo의 특정 id를 입력하여 삭제
